# material-shapes-ts

Type-safe implementation of Material Shapes for the web, ported from the AndroidX Graphics Shapes library.

This library provides a set of tools for creating and manipulating rounded polygonal shapes, ideal for creating adaptive UI components, smooth transitions, and complex clipping paths.

## Installation

```bash
npm install material-shapes-ts
# or
pnpm add material-shapes-ts
# or
yarn add material-shapes-ts
```

## Key Use Cases

### Clipping an Image to a Shape in a Vue Application

You can use the shapes generated by this library to clip images in your Vue application. Here's an example of how to clip an image to an 8-leaf clover shape:

```vue
<template>
  <div class="clover-container">
    <img src="your-image.jpg" class="clover-image" />
  </div>
</template>

<script setup lang="ts">
  import { onMounted } from 'vue';
  import { MaterialShapes, roundedPolygonToPath } from 'material-shapes-ts';

  onMounted(() => {
    const clover8Leaf = MaterialShapes.Clover8Leaf;
    const path = roundedPolygonToPath(clover8Leaf);
    const container = document.querySelector('.clover-image') as HTMLElement;
    if (container) {
      container.style.clipPath = `path('${path.toSvgPathData()}')`;
    }
  });
</script>

<style scoped>
  .clover-container {
    width: 200px;
    height: 200px;
  }

  .clover-image {
    width: 100%;
    height: 100%;
    object-fit: cover;
    /* Initial clip-path or fallback could go here */
  }
</style>
```

### Inserting a Shape into a Vue Application

You can also use this library to insert shapes directly into your Vue application as SVG elements. Here's an example of how to insert a bun shape:

```vue
<template>
  <svg width="100" height="100" viewBox="0 0 1 1">
    <path :d="bunPath" fill="saddlebrown" />
  </svg>
</template>

<script setup lang="ts">
  import { ref } from 'vue';
  import { MaterialShapes, roundedPolygonToPath } from 'material-shapes-ts';

  const bun = MaterialShapes.Bun;
  const path = roundedPolygonToPath(bun);
  const bunPath = ref(path.toSvgPathData());
</script>
```

### Morphing Between Shapes in a Vue Application

This library also supports morphing between shapes. Here's an example of how to morph from a circle to a heart, and then to a 4-leaf clover:

```vue
<template>
  <svg width="100" height="100" viewBox="0 0 1 1">
    <path :d="morphPath" fill="deeppink" />
  </svg>
</template>

<script setup lang="ts">
  import { ref, onMounted } from 'vue';
  import { MaterialShapes, Morph, morphToPath } from 'material-shapes-ts';

  const morphPath = ref('');

  onMounted(() => {
    const circle = MaterialShapes.Circle;
    const heart = MaterialShapes.Heart;
    const clover4Leaf = MaterialShapes.Clover4Leaf;

    const morph1 = new Morph(circle, heart);
    const morph2 = new Morph(heart, clover4Leaf);

    let progress = 0;
    let morph = morph1;

    function animate() {
      progress += 0.01;
      if (progress > 1) {
        progress = 0;
        morph = morph === morph1 ? morph2 : morph1;
      }
      const path = morphToPath(morph, progress);
      morphPath.value = path.toSvgPathData();
      requestAnimationFrame(animate);
    }

    animate();
  });
</script>
```
